Create a custom error generator

Parameters:

  • name must be a String.
    (The error name)

  • parameters must be an Object.
    (An optional set of key=value pairs to attach to errors)

  • Parent must be a Function.
    (An optional parent Error class to inherit from)

  • Constructor must be a Function.
    (An optional constructor to inherit additional properties from)

Returns a Function
